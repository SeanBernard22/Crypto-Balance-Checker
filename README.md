# Crypto-Balance-Checker-addresses
This software retrieves Bitcoin, Ethereum, Binance Coin, Solana, XRP, Cardano, Dogecoin, Avalanche, Polkadot, Tron, Chainlink, Polygon, Litecoin, Stellar, Uniswap, Monero, Shiba Inu, Cosmos, Aptos, Near Protocol, Ethereum Classic, Hedera, VeChain, Maker, Arbitrum, Optimism, Filecoin, Algorand, Aave, Tezos balances from one or more  addresses listed in the Crypto_addresses.txt file. The software returns a table with the  balance and current value in USDT and EUR for each address, the total amounts are also calculated (current Crypto price is retrieved from https://binance.com)
